# visualCountDown

a node that will display a countdown on screen

## Installation

Make sure to download the ChillCube Developer tools to use this addon: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Linux
To add this to your project, copy paste these commands into terminal at the root of your project:
```bash
clone-gd-addon https://github.com/ChillCube/visualCountDown.git
```

## Documentation
You can find the documentation at: https://github.com/ChillCube/visualCountDown/blob/main/DOCUMENTATION.md

## Deinstallation
### Linux
Run this in the root of your godot project:
```bash
remove-gd-addon visualCountDown
```

## For Developers
### Push Changes
Run this at the root of your directory to push changes to any addons you have:
```bash
push-all-addons
```
